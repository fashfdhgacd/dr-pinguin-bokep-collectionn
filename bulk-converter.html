Bulk Video Converter

Paste judul + link (format sederhana) → otomatis jadi JSON siap pakai

Cara pakai:
1. Paste judul di baris pertama
2. Paste link embed di baris kedua
3. Kosongkan 1 baris
4. Ulangi untuk video berikutnya

Tool ini otomatis mendeteksi IndoAV atau UserBokep dan membuat direct link.
